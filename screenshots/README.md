# Screenshots

This folder is used to store testing evidence, bug screenshots, and UI validation images.

## Example Files

```txt
bug-example.png
login-page.png
api-response.png
```

## Usage

Screenshots can be used as evidence in bug reports or testing summary documents.

Example:

```txt
/screenshots/bug-example.png
```

## Notes

Do not include sensitive user data in screenshots.
All screenshots should be used only for testing evidence and documentation.
